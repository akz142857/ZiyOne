四册系统规划

不要把所有知识挤进一本书。建议拆成 4 本小绘本，每本一个主题，重复同一套学习格式，降低认知负担。

Book 01

Tomi Counts the Stars
数字 Numbers

从 one 到 twenty，再扩展到 thirty、forty、one hundred。核心句型：I see… / How many?

Book 02

What Time Is It, Tomi?
时间 Time

学习整点、半点、morning / afternoon / evening / night。核心句型：What time is it?

Book 03

Tomi’s Week
星期 Days

学习 Monday 到 Sunday，today / tomorrow / yesterday。核心句型：Today is Monday.

Book 04

Tomi’s Year
月份 Months

学习 January 到 December，四季和生日表达。核心句型：My birthday is in May.

三层拼读系统

以自然拼读为主：看字母组合、音节切分、常见发音规律，并配合标准音标。拼读提示不使用中文谐音，避免形成错误依赖。

1. 中文意思

快速建立理解：一、七点、星期一、一月。

2. English

大字号呈现核心单词或短语：one, seven o’clock, Monday, January。

3. 音标 + 拼读

保留标准音标，同时拆分词块：Mon + day, Jan-u-ar-y。

4. 例句输入

用短句进入真实表达：I see one star. / Today is Monday.
